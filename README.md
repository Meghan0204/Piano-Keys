## How It Works

1. **HTML** defines the piano keys and layout.  
2. **CSS** styles the key elements and layout to resemble a piano keyboard.  
3. **JavaScript** listens for keyboard events and click events to play corresponding audio files.  
4. Sound files are mapped to specific keys so pressing a key or clicking its onscreen counterpart plays the correct note.

This project demonstrates practical use of **DOM manipulation**, **event listeners**, and **audio playback** in a browser context.
